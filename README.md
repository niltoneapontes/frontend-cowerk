# Frontend Cowerk 

[![Netlify Status](https://api.netlify.com/api/v1/badges/5f7ae222-410d-43ff-9636-c366e077246f/deploy-status)](https://app.netlify.com/sites/cowerk/deploys)

Front-end da CoWerk gem, uma aplicação web que gerencia um espaço de coworking.

O back-end desta aplicação pode ser encontrado em https://github.com/niltoneapontes/backend-cowerk.

A aplicação pode ser acessada em:
https://cowerk.netlify.app/
(pode apresentar instabilidade devido ao mLab MongoDB)

# Utilizando o back-end
Para utilizar esta aplicação é necessário realizar um "git clone" e em seguida rodar o comando "npm install" para instalar as dependências necessárias. Em seguida, utilize o comando "npm start" para iniciar o servidor.
